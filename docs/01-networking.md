# Networking Fundamentals

## Overview

This module covers the essential networking concepts that form the foundation of cybersecurity. Understanding how networks operate is crucial for identifying vulnerabilities and implementing security measures.

## Learning Objectives

By the end of this module, you will be able to:
- Understand the OSI and TCP/IP models
- Identify common network protocols and their security implications
- Recognize network topologies and their security considerations
- Understand basic network security concepts

## Topics Covered

### 1. Network Models
- OSI 7-Layer Model
- TCP/IP Model
- Protocol Stack Interaction

### 2. Core Protocols
- TCP vs UDP
- HTTP/HTTPS
- DNS and DNS Security
- DHCP
- ARP

### 3. Network Infrastructure
- Routers and Switches
- Firewalls
- VPNs
- Network Segmentation

### 4. IP Addressing and Subnetting
- IPv4 and IPv6
- CIDR Notation
- Private vs Public IP Ranges
- NAT and Port Forwarding

## Security Implications

- Man-in-the-Middle Attacks
- Protocol Vulnerabilities
- Network Reconnaissance
- Traffic Analysis
