# Simple Cybersecurity

A comprehensive educational repository for learning cybersecurity fundamentals through structured documentation and hands-on laboratory exercises.

## üéØ Overview

This repository provides a structured approach to learning cybersecurity, covering essential topics from networking fundamentals to advanced security practices. It's designed for beginners and intermediate learners who want to develop practical cybersecurity skills through theoretical knowledge and hands-on experience.

## üöÄ Getting Started

### Prerequisites

- Basic understanding of computer networks and operating systems
- Access to a computer with virtualization capabilities (8GB+ RAM recommended)
- Willingness to learn and practice ethical hacking principles

### Learning Path

1. **Start with Theory**: Read through the documentation modules in order
2. **Set Up Your Lab**: Complete [Lab 00: Setup](docs/labs/lab-00-setup.md) to configure your environment
3. **Practice Hands-On**: Work through the laboratory exercises
4. **Apply Knowledge**: Participate in CTF challenges and practice scenarios
5. **Stay Current**: Follow cybersecurity news and continue learning

### Quick Start

```bash
# Clone the repository
git clone https://github.com/clintoo/simple-cybersec.git
cd simple-cybersec

# Start with the documentation
# Read docs/01-networking.md through docs/07-legal-and-ethics.md

# Set up your lab environment
# Follow docs/labs/lab-00-setup.md

# Begin hands-on practice
# Complete labs in order: lab-01, lab-02, lab-03, then lab-ctf-overview
```

## üõ°Ô∏è Ethical Guidelines

**Important**: This repository is for educational purposes only. All techniques and tools should only be used on systems you own or have explicit permission to test.

### Core Principles

- **Only test authorized systems** - Never test systems without explicit permission
- **Respect privacy and data** - Protect any sensitive information encountered
- **Follow responsible disclosure** - Report vulnerabilities through proper channels
- **Comply with laws** - Understand and follow applicable cybersecurity laws
- **Use knowledge defensively** - Focus on improving security, not causing harm
